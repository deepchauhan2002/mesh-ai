## Usage

Set your OpenAI API Key in `.env`

```bash
git clone
echo "<your_key>" > .env
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun index.ts
```

This project was created using `bun init` in bun v0.5.7. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
